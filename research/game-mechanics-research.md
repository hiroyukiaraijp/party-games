# 知育パーティゲーム メカニクス調査・提案書

**調査日**: 2026-03-20
**前提**: Webブラウザ（スマホ対応静的サイト）、2〜15人、スマホ1台共有、知育×パーティ感

---

## 1. 言語力・語彙力

### 1-1. 爆速しりとりバトル（Speed Shiritori Battle）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 制限付きしりとりを早押し形式で行う。毎ラウンド「3文字以上」「食べ物のみ」「濁音で始まる」等のランダム制約が付与される |
| **詳細な遊び方** | 画面に制約条件が表示される → タイマー開始 → プレイヤーは順番に端末に単語を入力 → 制約違反・既出・時間切れで脱落 → 最後まで生き残った人が勝利 |
| **鍛える認知能力** | 語彙検索速度、制約下での思考（実行機能）、ワーキングメモリ |
| **プレイ人数・所要時間** | 2〜15人 / 1ラウンド2〜3分 |
| **盛り上がりポイント** | 制約が毎回変わるので飽きない。タイマーのプレッシャーで焦る姿が面白い。「え、それ食べ物!?」という突っ込みが発生 |
| **難易度調整** | 制約の数（1個→複数同時）、文字数制限、タイマー秒数、使用禁止文字の追加 |
| **実装難易度** | 低 — テキスト入力＋タイマー＋ルール判定のみ。辞書APIは不要（プレイヤー同士の合意制でも可） |
| **類似既存ゲーム** | ワードバスケット、限界しりとりMobile |

### 1-2. カタカナ禁止ジェスチャー（ボブジテン・パーティ）

| 項目 | 内容 |
|------|------|
| **ルール概要** | お題のカタカナ語を「カタカナを一切使わず」に説明し、他プレイヤーに当ててもらう |
| **詳細な遊び方** | 出題者がスマホで秘密のお題を確認（例:「シャンプー」）→ カタカナ語を使わず口頭で説明 → 他プレイヤーが早押しで回答入力 → 正解者と出題者に得点 → 出題者を順番に交代 |
| **鍛える認知能力** | 言い換え能力（語彙の柔軟性）、表現力、抑制制御（カタカナを言いそうになるのを抑える） |
| **プレイ人数・所要時間** | 3〜15人 / 1ラウンド1〜2分、全体10〜15分 |
| **盛り上がりポイント** | つい「カタカナ」を言ってしまい爆笑。説明が下手すぎて意味不明になる展開が最高 |
| **難易度調整** | お題の難易度（日常語→専門用語）、追加制約（ジェスチャー禁止、「〜的な」禁止等） |
| **実装難易度** | 低 — お題データベース＋表示のみ。判定はプレイヤー同士で行う |
| **類似既存ゲーム** | ボブジテン、タブー |

### 1-3. 数字deコトバ（ito パーティ）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 1〜100の数字を「テーマに沿った言葉」で表現し、全員の数字を小さい順に並べる協力ゲーム |
| **詳細な遊び方** | テーマ発表（例:「怖いもの」）→ 各プレイヤーにランダムな数字（1〜100）が秘密配布 → 自分の数字をテーマに沿った言葉で表現（例: 37なら「暗い廊下」、85なら「心霊写真」）→ 全員で相談して小さい順に並べる → 正しい順なら成功 |
| **鍛える認知能力** | 抽象的思考、スケール感覚、他者の価値観理解、コミュニケーション能力 |
| **プレイ人数・所要時間** | 3〜15人 / 1ラウンド5分 |
| **盛り上がりポイント** | 価値観の違いが浮き彫りに。「え、それ85!? 俺の80より怖くない!」で大盛り上がり |
| **難易度調整** | 数字の範囲（1〜50→1〜100）、テーマの抽象度、許容誤差の設定 |
| **実装難易度** | 低 — 数字ランダム配布＋テーマ表示＋並べ替えUI |
| **類似既存ゲーム** | ito（イト）、Wavelength |

### 1-4. ワード連想チェーン（Word Chain Blast）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 起点ワードから連想で繋ぎ、最終的にゴールワードに到達するまでのチェーンの短さを競う |
| **詳細な遊び方** | 画面に起点（例:「海」）とゴール（例:「電車」）が表示 → 各プレイヤーが連想チェーンを考えて入力（例: 海→魚→市場→築地→電車 = 4ステップ）→ 最短チェーンのプレイヤーが発表 → 他プレイヤーが「連想として妥当か」を投票判定 → 最短かつ承認されたチェーンが勝利 |
| **鍛える認知能力** | 意味ネットワークの柔軟性、発散的思考、創造性 |
| **プレイ人数・所要時間** | 2〜15人 / 1ラウンド3分 |
| **盛り上がりポイント** | 無理矢理な連想に「それはアリ?ナシ?」の議論が白熱。思いもよらない連想ルートに感心 |
| **難易度調整** | 起点とゴールの意味的距離、制限時間、「名詞のみ」等の品詞制約 |
| **実装難易度** | 低 — ワードペア出題＋テキスト入力＋投票UI |
| **類似既存ゲーム** | Codenames（連想要素）、Wikipedia Game |

### 1-5. 頭文字バトルロイヤル（Initial Battle）

| 項目 | 内容 |
|------|------|
| **ルール概要** | ランダムに指定された頭文字＋カテゴリで、他の人と被らない単語を制限時間内に書く |
| **詳細な遊び方** | 画面に「か」×「動物」のように頭文字とカテゴリが表示 → 全員が同時にスマホに回答入力 → 一斉公開 → 他の人と被った回答は0点、ユニークな回答のみ得点 → 5ラウンドで合計点を競う |
| **鍛える認知能力** | 語彙の広さ、カテゴリ流暢性（前頭葉機能）、戦略的思考（被りを避ける） |
| **プレイ人数・所要時間** | 3〜15人 / 1ラウンド1分、全体10分 |
| **盛り上がりポイント** | 「かめ」「かば」が大量に被って全滅する展開。マイナーな回答が称賛される |
| **難易度調整** | カテゴリの広さ、頭文字の難しさ（「ぬ」等）、回答数の制限 |
| **実装難易度** | 低 — 同時入力＋重複判定＋スコア計算 |
| **類似既存ゲーム** | Scattergories、カテゴリーズ |

### 1-6. 逆引き辞書ゲーム（意味から引け!）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 辞書の「意味説明文」だけが表示され、その単語を当てるクイズ |
| **詳細な遊び方** | 画面に辞書風の定義文が表示（例:「四本脚の家具で、上面が平らで食事などに使う」）→ プレイヤーが早押しで回答 → 正解で得点。不正解はペナルティ → 徐々にヒントが追加される |
| **鍛える認知能力** | 語彙の深さ、定義理解力、推論能力 |
| **プレイ人数・所要時間** | 2〜15人 / 1問30秒、全体10分 |
| **盛り上がりポイント** | 簡単そうで意外と出てこない。ヒントが出るたびに「あー!」の声 |
| **難易度調整** | 単語の頻度（日常語→難語）、定義文の抽象度、ヒント段階の数 |
| **実装難易度** | 中 — 質の高い定義文データの準備が必要 |
| **類似既存ゲーム** | たほいや、Balderdash |

## 2. 処理速度・反応力

### 2-1. カラーパニック（Stroop Party）

| 項目 | 内容 |
|------|------|
| **ルール概要** | ストループ効果を利用。表示される「色の名前」の「文字の色」を答える早押しバトル |
| **詳細な遊び方** | 画面に「あか」と青色で表示される → 正解は「青」→ プレイヤーは順番にスマホを回して素早くタップ → 正解＋反応時間でスコア計算 → 10問ずつ → 全員終了後にランキング表示 |
| **鍛える認知能力** | 抑制制御（干渉抑制）、処理速度、注意の切り替え |
| **プレイ人数・所要時間** | 2〜15人 / 1人30秒、全体5〜10分 |
| **盛り上がりポイント** | 簡単そうなのに脳がバグる。間違えたときの「えーーー!」が最高。タイムアタック形式で競争心を煽る |
| **難易度調整** | 表示速度、選択肢の数（2色→5色）、逆ストループ（色を文字で答える）の追加 |
| **実装難易度** | 低 — 色付きテキスト表示＋タップ判定＋タイマー |
| **類似既存ゲーム** | Impulse（アプリ）、Brain Spark |

### 2-2. フラッシュ暗算バトル（Flash Math Duel）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 画面に高速で数字が表示され、合計を計算して回答。精度と速度で競う |
| **詳細な遊び方** | 3〜5個の数字が0.5秒間隔で画面に表示 → 消えた後に合計を入力 → 正解＋速度でスコア → ラウンドごとに数字の桁数・個数が増加 → プレイヤー交代で全員チャレンジ |
| **鍛える認知能力** | 暗算能力、ワーキングメモリ、処理速度 |
| **プレイ人数・所要時間** | 2〜15人 / 1人1分、全体5〜15分 |
| **盛り上がりポイント** | 「見えなかった!」「4桁とか無理!」の悲鳴。得意な人のドヤ顔と苦手な人の絶望感の対比 |
| **難易度調整** | 数字の桁数（1桁→3桁）、表示個数（3→7）、表示速度、演算の種類（加算→加減算混合） |
| **実装難易度** | 低 — 数字アニメーション＋入力フォーム＋スコア計算 |
| **類似既存ゲーム** | フラッシュ暗算アプリ各種 |

### 2-3. タップ de サバイバル（Tap Survivor）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 画面の指示に素早く反応するミニゲーム連続。間違えたら脱落のサバイバル形式 |
| **詳細な遊び方** | 全員で画面を囲む → 「赤が出たらタップ!」「音が鳴ったらタップ!」等の指示 → ランダムにフェイント（緑が出る、似た音が鳴る等）→ フェイントでタップしたら脱落 → 最後の1人が勝利 |
| **鍛える認知能力** | Go/No-Go課題（抑制制御）、選択的注意、反応速度 |
| **プレイ人数・所要時間** | 2〜10人（画面タッチ同時） / 3〜5分 |
| **盛り上がりポイント** | フェイントに引っかかる人続出。「押すな押すな」の緊張感。脱落者が応援側に回る |
| **難易度調整** | フェイントの頻度・巧妙さ、反応猶予時間、同時判定の数 |
| **実装難易度** | 中 — マルチタッチ判定、同時プレイ時のタッチ位置管理が必要 |
| **類似既存ゲーム** | Red Light Green Light系ゲーム、Simon Says |

### 2-4. 瞬間カウンター（Flash Counter）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 画面に一瞬だけ表示されるオブジェクトの数を正確に数える。サビタイジング能力を競う |
| **詳細な遊び方** | 画面にランダムな数のドット/図形が0.5秒だけ表示 → 何個あったかを入力 → 正解で得点、ラウンドが進むと数が増加（5個→15個） → プレイヤー交代で実施 |
| **鍛える認知能力** | サビタイジング（瞬間的数量把握）、視覚処理速度、注意力 |
| **プレイ人数・所要時間** | 2〜15人 / 1人1分、全体5〜15分 |
| **盛り上がりポイント** | 「7!...いや8!?」の迷い。意外と当たる人と全然ダメな人の差が面白い |
| **難易度調整** | 表示時間（1秒→0.3秒）、オブジェクト数、紛らわしい図形の混入、色別カウント |
| **実装難易度** | 低 — ランダム配置描画＋タイマー＋入力判定 |
| **類似既存ゲーム** | Human Benchmark（Number Memory）、脳トレアプリ各種 |

### 2-5. 逆さま指令（Reverse Command）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 画面の指示と「逆」のことをする反射神経ゲーム。「右」と出たら左をタップ |
| **詳細な遊び方** | 画面に矢印や指示テキストが表示 → プレイヤーは指示と逆の操作を実行（「上スワイプ」→下にスワイプ）→ 正解＋反応時間でスコア → 指示の種類が徐々に増加（方向→色→数字の大小） |
| **鍛える認知能力** | 認知的柔軟性、抑制制御、タスクスイッチング |
| **プレイ人数・所要時間** | 2〜15人 / 1人30秒、全体5〜10分 |
| **盛り上がりポイント** | 脳がフリーズする瞬間が面白い。慣れてきたところでルール追加される鬼畜仕様 |
| **難易度調整** | 指示の種類数、切り替え頻度、「逆」と「そのまま」がランダムに混在 |
| **実装難易度** | 低 — 指示表示＋スワイプ/タップ判定 |
| **類似既存ゲーム** | Simon Says（逆版）、Brain Age系 |

### 2-6. 神経衰弱スプリント（Memory Sprint）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 超高速神経衰弱。カードを覚える時間は最初の3秒だけ。その後はスピード勝負 |
| **詳細な遊び方** | 全カードが3秒間オープン → 一斉に裏返し → プレイヤーが交代で2枚めくる → ペア成立で得点＋連続ターン → 制限時間内に最多ペアを獲得した人が勝利 |
| **鍛える認知能力** | 視覚的短期記憶、空間記憶、処理速度 |
| **プレイ人数・所要時間** | 2〜8人 / 1ゲーム3〜5分 |
| **盛り上がりポイント** | 最初の3秒の必死さ。連続正解の爽快感。カード数を増やすと記憶力の限界が見える |
| **難易度調整** | カード枚数（8枚→24枚）、覚える時間（5秒→1秒）、絵柄の類似度 |
| **実装難易度** | 低 — カードグリッド＋フリップアニメーション＋マッチ判定 |
| **類似既存ゲーム** | 神経衰弱、Concentration |

## 3. ワーキングメモリ・記憶力

### 3-1. 数列リレー（Number Relay）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 表示される数列を記憶して正確に再現する。プレイヤーごとに数列が1つずつ長くなる |
| **詳細な遊び方** | Player1: 3桁の数列が2秒表示 → 入力 → 正解ならPlayer2へ → Player2は4桁 → ... → 間違えた人が脱落 → 最後まで正解し続けた人が勝利 |
| **鍛える認知能力** | 数字スパン（短期記憶容量）、注意集中、符号化戦略 |
| **プレイ人数・所要時間** | 2〜15人 / 5〜10分 |
| **盛り上がりポイント** | 桁数が増えるにつれ「もう無理!」の声。意外な人が記憶力王になる |
| **難易度調整** | 開始桁数、増加ペース、表示時間、逆順再生モード |
| **実装難易度** | 低 — 数列生成＋表示＋入力照合 |
| **類似既存ゲーム** | Digit Span Test、Human Benchmark（Sequence Memory） |

### 3-2. 変化を見つけろ!（Spot the Change）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 画面の絵/写真が一瞬消えて再表示。何が変わったかを当てる |
| **詳細な遊び方** | 画面にイラスト（10〜20個のオブジェクト）が5秒間表示 → 1秒間ブランク → 1〜3箇所変更して再表示 → プレイヤーが変化した箇所をタップ → 早く全箇所見つけた人が勝利 |
| **鍛える認知能力** | 変化検出（視覚的ワーキングメモリ）、注意の広さ、詳細観察力 |
| **プレイ人数・所要時間** | 2〜15人 / 1問1分、全体10分 |
| **盛り上がりポイント** | 「どこ!? 全然わからん!」のパニック。見つけた人のドヤ顔。変化が微妙すぎる問題で全員苦戦 |
| **難易度調整** | オブジェクト数、変化の微妙さ（色が少し違う→物が消える）、ブランク時間 |
| **実装難易度** | 中 — 画像生成/管理、変化パターンのプログラム的生成が必要 |
| **類似既存ゲーム** | 間違い探し、Change Blindness実験 |

### 3-3. オーダー記憶（Order Memory Challenge）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 画面に表示されるアイテムの「順番」を記憶して再現する。Simonゲームの進化版 |
| **詳細な遊び方** | 画面上の6つのパネルが順番に光る → プレイヤーは同じ順番でタップ → 正解なら次のラウンドで1ステップ追加 → 間違えたら脱落。プレイヤー交代制で、各自の最高記録を競う |
| **鍛える認知能力** | 系列的ワーキングメモリ、パターン認識、手続き記憶 |
| **プレイ人数・所要時間** | 2〜15人 / 1人1〜2分、全体5〜15分 |
| **盛り上がりポイント** | 「3つ目どこだっけ!?」の焦り。長い系列を覚える人への尊敬。脱落者からの野次 |
| **難易度調整** | パネル数（4→9）、光る速度、音の有無、色＋位置の二重記憶 |
| **実装難易度** | 低 — パネルアニメーション＋タップ順序判定 |
| **類似既存ゲーム** | Simon、Human Benchmark（Sequence Memory） |

### 3-4. 買い物リスト暗記バトル（Shopping Memory）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 「買い物リスト」を順番に1つずつ追加して全員で暗記。言えなくなったら脱落 |
| **詳細な遊び方** | Player1「りんご」→ Player2「りんご、牛乳」→ Player3「りんご、牛乳、パン」→ ... → リストをすべて順番通りに言えなかったら脱落 → スマホは正解リストの記録と判定に使用 |
| **鍛える認知能力** | エピソード記憶、系列記憶、記憶術（チャンキング・物語法）の活用 |
| **プレイ人数・所要時間** | 3〜15人 / 5〜10分 |
| **盛り上がりポイント** | 10個超えたあたりから崩壊が始まる。変なアイテムほど覚えやすい皮肉。記憶術を駆使する人が出現 |
| **難易度調整** | カテゴリ制限（食品のみ→何でもOK）、追加アイテム数（1個→2個ずつ）、逆順モード |
| **実装難易度** | 低 — リスト管理＋音声入力 or テキスト入力＋照合 |
| **類似既存ゲーム** | I went to market...（英語圏の定番記憶ゲーム） |

### 3-5. ペア探しタイムアタック（Pair Hunt）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 大量のカードの中から指定されたペアを記憶力だけで見つけるタイムアタック |

## 4. 社会的知性・読心術

### 4-1. ワードウルフ（Word Wolf）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 全員に似たお題が配られるが、1人だけ違うお題の「ウルフ」。会話でウルフを見つけ出す |
| **詳細な遊び方** | スマホが各プレイヤーに秘密のお題を表示（例: 多数派「りんご」/ウルフ「みかん」）→ お題について自由に会話（3分）→ 投票でウルフを指名 → ウルフが当てられたら市民勝利、逃げ切ったらウルフ勝利 |
| **鍛える認知能力** | 社会的推論、情報の非対称性の理解、言語的曖昧さの操作、観察力 |
| **プレイ人数・所要時間** | 3〜15人 / 1ラウンド5分 |
| **盛り上がりポイント** | 「これ同じお題の人の発言?」の疑心暗鬼。ウルフが上手く紛れると大盛り上がり |
| **難易度調整** | お題の類似度（りんご/みかん → りんご/バナナ）、ウルフの人数、会話時間 |
| **実装難易度** | 低 — お題配布＋タイマー＋投票UI |
| **類似既存ゲーム** | ワードウルフ（どこパ等）、The Chameleon |

### 4-2. 空気読みスケール（Vibes Check）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 主観的な質問に対して全員がスケール（1〜100）で回答。「多数派の答え」に近い人が勝利 |
| **詳細な遊び方** | 質問表示（例:「カレーの辛さは何辛が一番美味しい? 1=甘口、100=激辛」）→ 全員がスライダーで回答 → 一斉公開 → 全員の中央値に最も近い人が得点 → 最も外れた人にペナルティ |
| **鍛える認知能力** | 社会的認知（多数派予測）、メタ認知、集団の傾向理解 |
| **プレイ人数・所要時間** | 3〜15人 / 1問1分、全体10分 |
| **盛り上がりポイント** | 価値観の違いが数字で可視化される衝撃。「お前100!?」の驚き。自分が少数派だと知るショック |
| **難易度調整** | 質問の主観性の度合い、スケールの粒度、「平均」vs「中央値」vs「最頻値」の変更 |
| **実装難易度** | 低 — スライダーUI＋集計＋結果表示 |
| **類似既存ゲーム** | Wavelength、ito（数値表現部分） |

### 4-3. 嘘つきは誰だ（Liar Detector）

| 項目 | 内容 |
|------|------|
| **ルール概要** | 1人が嘘のエピソードを話し、他の人は本当のエピソードを話す。質問で嘘つきを見破る |
| **詳細な遊び方** | スマホがランダムに1人を「嘘つき」に指定 → お題発表（例:「小学生の頃の思い出」）→ 全員が1つエピソードを発表（嘘つきは作り話）→ 質問タイム（2分）→ 投票 → 嘘つきを当てたら市民勝利 |
| **鍛える認知能力** | 嘘検出能力、質問設計力、非言語コミュニケーションの読み取り、演技力 |
| **プレイ人数・所要時間** | 4〜15人 / 1ラウンド5〜8分 |
| **盛り上がりポイント** | 嘘つきの動揺。鋭い質問で追い詰める展開。意外な人が嘘が上手い発見 |
| **難易度調整** | お題の具体性、嘘つきの人数、質問時間、嘘つきに事前準備時間を与えるか |
| **実装難易度** | 低 — 役割ランダム割り当て＋タイマー＋投票UI |
| **類似既存ゲーム** | 2 Truths 1 Lie、人狼系ゲーム |
