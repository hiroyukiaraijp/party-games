# Party Games サイト企画

## 1. サイト名候補 & デザイン方向性

### サイト名候補
| 名前 | コンセプト |
|------|----------|
| **アタマジム (AtamaGym)** | 頭のジム＝脳を鍛える場所。ポップで覚えやすい |
| **ブレインパーティ (BrainParty)** | 脳トレ×みんなで楽しむパーティ感 |
| **ヒラメキ! (Hirameki!)** | 「ひらめき」＝直感・創造力。日本語の響きが良い |
| **ノビノビ (NobiNobi)** | 伸び伸び＝能力が伸びる＋楽しく遊ぶ |
| **アソビラボ (AsobiLab)** | 遊びの研究所。実験的・知的な雰囲気 |

### デザイン方向性
- **カラー**: ダーク系から脱却 → パステル系/ネオンポップ（黄・ピンク・水色・紫）
- **フォント**: 丸ゴシック系で柔らかさ。タイトルは太め＆遊び心
- **テイスト**: ゲームセンター × 知育 × フレンドリー
- **アイコン**: 各ゲームにポップなイラスト風アイコン
- **アニメーション**: ボタンにバウンス、正解時にパーティクル、軽い演出

## 2. 知育・脳トレ コンテクスト設計

### A. 認知能力カテゴリ（脳科学ベース）

| カテゴリ | 鍛える力 | ゲーム例 |
|---------|---------|---------|
| **言語力・語彙力** | 語彙検索速度、言語連想 | ワードスロット(実装済)、しりとりバトル、連想ゲーム |
| **処理速度** | 情報処理の速さ、反応時間 | 早押しクイズ、フラッシュ暗算、色文字テスト(ストループ) |
| **ワーキングメモリ** | 短期記憶、情報の保持と操作 | 神経衰弱、数列記憶、逆唱チャレンジ |
| **注意力・集中力** | 選択的注意、持続的注意 | 間違い探し、変化検出、マルチタスクゲーム |
| **論理的思考** | 推理、パターン認識 | 数独バトル、論理パズル、暗号解読 |
| **創造力・発想力** | 拡散的思考、柔軟性 | お題大喜利、逆しりとり、制約付き創作 |
| **空間認識** | 図形認知、心的回転 | タングラム、図形パズル、鏡文字 |
| **社会的知性** | 読心、共感、駆け引き | 人狼ライト、嘘つきゲーム、表情読み |

### B. 特性育成マッピング

| 育てたい特性 | 対応する認知能力 | ゲームタイプ |
|------------|---------------|------------|
| **頭の回転の速さ** | 処理速度 + 語彙力 | 制限時間付き言語ゲーム |
| **記憶力** | ワーキングメモリ | 記憶系チャレンジ |
| **集中力** | 注意力 | 持続注意ゲーム |
| **ひらめき力** | 創造力 + 論理思考 | なぞなぞ・パズル系 |
| **コミュ力・読心術** | 社会的知性 | 対人推理ゲーム |
| **冷静さ・判断力** | 論理思考 + 注意力 | プレッシャー下の意思決定ゲーム |

### C. スマホ1台テーブルゲーム パターン

| パターン | 遊び方 | 例 |
|---------|-------|-----|
| **画面共有・早い者勝ち** | 1台を全員で見て叫ぶ | ワードスロット(実装済) |
| **回し読み型** | 画面を順番に見て入力 | 人狼ライト、暗号伝言 |
| **司会者型** | 1人がスマホ操作、他は口頭参加 | ジェスチャー、クイズ出題 |
| **同時入力型** | 順番に秘密入力→一斉公開 | 数字バトル、格付けチェック |
| **タイマー型** | 制限時間内に全員でクリア | 協力パズル、ボム回し |

## 3. Gemini Deep Research への指示

以下を Gemini の Deep Research に投げてください:

---

**リサーチテーマ: 知育・脳トレ パーティゲームの設計エビデンスと既存事例の網羅調査**

以下の5つの観点で、学術的エビデンスと実際のプロダクト事例を網羅的に調査してください。

### 調査項目

**1. 認知能力トレーニングの科学的根拠**
- 各認知能力（ワーキングメモリ、処理速度、注意力、言語力、論理思考、創造力、空間認識、社会的知性）のトレーニング効果に関する研究
- 「transfer effect（転移効果）」— ゲームで鍛えた能力が実生活に転移するかのエビデンス
- 年齢層別（子供、若年成人、高齢者）のトレーニング効果の違い
- Lumosity、Peak、BrainHQ等の既存脳トレアプリの効果検証論文

**2. マルチプレイヤー知育ゲームの設計パターン**
- 複数人で遊ぶ知育/脳トレゲームの既存事例（デジタル・アナログ両方）
- 競争型 vs 協力型のモチベーション効果の違い
- スマートフォン1台でのマルチプレイ設計パターンと成功事例
- Jackbox Games、QuizUp、Among Us等のマルチプレイ設計から学べること

**3. ゲーミフィケーションと継続率**
- 脳トレゲームのユーザー継続率を高める設計要素（スコアボード、ストリーク、レベルアップ等）
- 内発的動機付けを維持するメカニズム
- 「フロー状態」を誘発するゲーム難易度設計

**4. 認知特性の診断・プロファイリング**
- ゲームプレイデータから認知特性を推定する手法（既存研究・プロダクト）
- 各認知能力を測定するのに適したタスクの種類と信頼性
- BIG5性格診断やMI理論（多重知能理論）とゲームパフォーマンスの関連研究
- プライバシーと倫理的配慮

**5. 具体的なゲームメカニクス案**
- 上記の各認知能力カテゴリごとに、スマホ1台・複数人で遊べるゲームメカニクスの提案
- 特に「言語」「処理速度」「ワーキングメモリ」「社会的知性」の4領域で5つずつ
- 各ゲームの想定プレイ時間、対象人数、難易度調整の方法

### 出力形式
- 各調査項目ごとにセクション分けし、引用元（論文タイトル・著者・年 or プロダクト名・URL）を明記
- 日本語で出力
- 最後に「すぐ実装可能なゲームアイデア TOP10」をランキング形式でまとめてください
