# 自己決定理論（SDT）のUI実装

## 概要
自律性・有能感・関係性の3つの基本的心理欲求を満たすための具体的なUI/UX実装パターンを定める。

## SDTの3要素

### 自律性（Autonomy）: 自分で選んでいる感覚
### 有能感（Competence）: 能力が向上している実感
### 関係性（Relatedness）: 他者との意味のある繋がり

## 自律性の実装パターン

### パターン1: 難易度の自己選択
- **実装**: 優秀な成績を収めた際に「チャレンジモード」への昇格を自分で選択できるオプション
- **NeuroNationの実績**: 18-30歳でコンバージョン率30%向上、30歳以上で14%向上
- **UI**: 正答率80%超の後に「次のレベルに挑戦する？」モーダルを表示
- **注意**: アルゴリズムが自動で難易度を上げるのとは別に、意識的な選択の余白を残す

### パターン2: ゲーム選択の自由度
- **実装**: ホーム画面で複数のゲームタイルを提示し、プレイヤーが自由に選択
- **UI**: カテゴリフィルター（言語/処理速度/記憶力/社会的知性）で絞り込み可能
- **注意**: 「おすすめ」は提示するが「強制」しない。「今日のおすすめ」バッジ付きで提案

### パターン3: カスタマイズ可能な設定
- **実装**: プレイ人数、ラウンド数、制限時間をゲーム開始前に調整可能
- **UI**: スライダーやステッパーで直感的に操作
- **デフォルト値**: 初心者にも適切なデフォルトを設定しつつ変更の余地を残す

## 有能感の実装パターン

### パターン1: 成長の可視化ダッシュボード（Insights機能）
- **Lumosityの方式**: 8回プレイで「ゲームごとの強み」解放、25回で「月間変動」可視化
- **UI要素**:
  - 折れ線グラフ: スコアの推移（過去7日/30日/全期間）
  - レーダーチャート: 8つの認知ドメインのバランス
  - 比較表示: 「前回比 +15%」のような差分ハイライト
  - マイルストーン: 「初めて○○を達成」のタイムライン表示
- **localStorage設計**: 各ゲームのスコア履歴を日付付きで保存

### パターン2: 即時フィードバックループ
- **正解時**: 緑のチェックマーク＋短い効果音（100ms以内に表示）
- **不正解時**: 赤のXマーク＋振動（hapticフィードバック）＋正解の表示
- **レベルアップ時**: フルスクリーンのお祝いアニメーション（1-2秒、スキップ可能）
- **注意**: フィードバックは瞬時。遅延は有能感を阻害する

### パターン3: 相対的な有能感の提示
- **実装**: スコアを「他のプレイヤーとの比較」で文脈化
- **UI**: 「同年代の上位○%です」（母集団がゲームユーザーであることを注記）
- **注意**: 絶対値だけでなく「成長率」も表示（今月のスコア上昇率等）

## 関係性の実装パターン

### パターン1: マルチプレイの体験記録
- **実装**: 誰と一緒にプレイしたかの履歴を保存
- **UI**: 「○○さんと5回プレイ」「一緒にプレイした仲間: 12人」
- **localStorage設計**: プレイヤー名とプレイ日時のログ

### パターン2: 協力達成の祝福
- **実装**: チーム/全員での目標達成時に特別な演出
- **UI**: 全員の名前が画面に並ぶ「Team Victory」画面
- **バッジ**: 「初めてのチームプレイ」「5人以上で協力クリア」等

### パターン3: リフレクションの共有
- **実装**: ゲーム終了後の結果をSNSシェア可能な画像で生成
- **UI**: プレイヤー名＋スコア＋ハイライトを含むビジュアルカード
- **canvas API**: ダイナミックにシェア画像を生成

### パターン4: 社会的アカウンタビリティ
- **Duolingo Friends Quest方式**: 毎回ランダムにペアを組み共同目標を設定
- **実装（ローカル版）**: 「今日の相棒」としてペアを表示、二人の合計スコアで報酬
- **効果**: フレンドフォロー者のコース修了率5.6倍（Duolingo実績）

## ドーパミン報酬系の設計
- **持続的（Tonic）活動**: 安定したベースラインの満足感 → 一貫したUI品質
- **位相的（Phasic）発火**: 予期せぬ報酬で探索行動を強化 → シークレットバッジ、ランダム報酬
- **設計原則**: 予測可能な報酬（ストリーク）と予測不能な報酬（シークレット）を混在させる

## SDT要素のUI配置マップ
```
[ホーム画面]
  - 自律性: ゲーム選択タイル（自由選択）
  - 有能感: ストリークカウンター、今日のスコアサマリー
  - 関係性: 「前回のプレイメンバー」表示

[ゲーム開始前]
  - 自律性: 人数/難易度/ラウンド数の設定
  - 関係性: プレイヤー名入力（顔文字アバター選択）

[ゲーム中]
  - 有能感: 即時フィードバック（正解/不正解の視覚+音声）
  - 自律性: ヒント使用の自己選択

[ゲーム終了後]
  - 有能感: スコア表示＋前回比＋レベル表示
  - 関係性: ランキング＋MVP投票＋シェア機能
  - 自律性: 「もう1回」or「別のゲームへ」の選択
```

## チェックリスト

- [ ] 有能感: 即時フィードバック（100ms以内）が実装されているか
- [ ] 有能感: 成長の可視化（グラフ、前回比）があるか
- [ ] 関係性: マルチプレイ時の共有体験が記録されるか
- [ ] 3要素がバランスよくUI内に配置されているか
