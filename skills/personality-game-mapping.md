# パーソナリティとゲームメカニクスの対応

## 概要
BIG5性格特性とゲーム内行動の相関、多重知能（MI）理論とメカニクスのマッピングを定める。

## BIG5 → ゲーム内行動パターン

### 1. 外向性（Extraversion）
- **高スコアのプレイスタイル**: 積極的探索、好戦的、ハイリスク・ハイリターン選好
- **行動指標**: 攻撃/アクションの総数、移動距離、探索率が高い
- **好むゲーム**: 早押し系、対戦系、表情アクターズ（表現系）
- **設計への含意**: 外向性が高い層にはスピード感と社会的露出の多いゲームを推薦

### 2. 誠実性（Conscientiousness）
- **高スコアのプレイスタイル**: 計画的、慎重なリソース管理、高い忍耐力
- **行動指標**: 失敗回数の少なさ、同一タスクの試行時間が長い（粘り強い）
- **好むゲーム**: 論理パズル、暗号解読、記憶力系
- **設計への含意**: 誠実性が高い層にはルールが明確で段階的に進むゲームを推薦

### 3. 神経症的傾向（Neuroticism）
- **高スコアのプレイスタイル**: 不規則なプレイ、失敗後の過剰反応、極端な回避行動
- **行動指標**: アクション頻度の分散が大きい、エラー後に反応速度が極端に低下
- **設計への含意**: フラストレーション耐性が低いため、DDAをより敏感に調整。失敗時のフィードバックを穏やかに

### 4. 開放性（Openness to Experience）
- **高スコアのプレイスタイル**: 好奇心旺盛、非定石の採用、多様なツールの実験
- **行動指標**: 使用アイテムの種類数が多い、行動ルートの多様性が高い
- **好むゲーム**: 創作系（ストーリー創作、フェイク辞書）、新しいルールのゲーム
- **設計への含意**: 新奇性と探索の余地を多く提供。定型的な反復は退屈を招く

### 5. 協調性（Agreeableness）
- **高スコアのプレイスタイル**: 利他的、チーム支援志向、公平な資源分配
- **行動指標**: 他者へのリソース譲渡回数、支援アクション数が多い
- **好むゲーム**: 協力型、チーム対抗型、価値観スケール
- **設計への含意**: 協調性が高い層には協力要素の強いゲームを推薦

## MI理論（8つの知能）→ ゲームメカニクス

### 1. 論理・数学的知能
- **メカニクス**: リソース管理、暗号解読、パターン発見パズル
- **データ**: ルール発見までの時間、効率的なリソース配分

### 2. 視覚・空間的知能
- **メカニクス**: 3Dナビゲーション、図形回転、迷路探索
- **データ**: 迷路脱出時間、操作エラー率

### 3. 身体・運動的知能
- **メカニクス**: タイミングアクション、精密タップ操作、反射神経
- **データ**: 反応速度（ms）、クリックの正確性

### 4. 音楽的知能
- **メカニクス**: リズム入力、聴覚シーケンス記憶
- **データ**: ピッチ認識の正答率、リズム同調性

### 5. 言語的知能
- **メカニクス**: 単語パズル、しりとり、NGワード、アナグラム
- **データ**: 語彙検索速度、指示の読解時間

### 6. 博物的知能
- **メカニクス**: カテゴリ分類、パターン認識、差異の発見
- **データ**: 分類の正確性と処理速度

### 7. 対人的知能
- **メカニクス**: ソーシャルディダクション、表情認識、交渉
- **データ**: 利他的選択の頻度、感情ラベルの正答率

### 8. 内省的知能
- **メカニクス**: 難易度の自己選択、戦略の自己修正
- **データ**: リスクテイクの変動、エラー後のリカバリー速度

## ゲーム推薦エンジンへの応用
```
ロジック:
1. プレイ履歴から好みのゲームタイプを分類
2. ゲームタイプとBIG5/MI理論の対応から傾向を推定
3. 「強み」を活かすゲームと「弱み」にチャレンジするゲームを両方提案

表示:
- 「あなたにおすすめ（得意分野）」: 成功体験を重ね有能感を維持
- 「新しいチャレンジ」: 普段プレイしないドメインのゲーム
```

## 注意事項
- BIG5やMIの推定は「傾向の示唆」であり「心理診断」ではない
- ユーザーにラベリング（「あなたは外向的です」等）を直接提示しない
- 代わりに「探索型プレイスタイル」「慎重型プレイスタイル」等のゲーム文脈の表現を使う
