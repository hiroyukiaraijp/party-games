# ゲーミフィケーションの行動心理学

## 概要
ストリーク、リーダーボード、バッジの3大ゲーミフィケーション要素の心理学的メカニズムと、負の側面の回避策を定める。

## 1. デイリーストリーク（連続記録）

### 心理学的メカニズム
- **損失回避（Loss Aversion）**: 同額の利益より損失の方が約2倍の精神的苦痛。積み上げた記録を失う恐怖が起動動機になる
- **ツァイガルニク効果**: 未完了タスクの方が記憶に残りやすい。ストリーク=常に「継続中」の状態
- **変動比率スケジュール**: ストリーク初期は50%の成長感（2→3日）、長期は0.5%（200→201日）。初期に強いドーパミン報酬

### 効果データ
- 7日ストリーク達成者: コース完了率3.6倍、翌日復帰率2.4倍
- ストリーク開始者: 長期エンゲージメント35%改善

### 負の側面: Streak Creep（ストリークの呪縛）
- 継続自体が目的化し、途切れた瞬間に燃え尽き→二度と戻らない
- **対策: ストリークフリーズ**
  - 1日の失敗を保護するアイテム（自動or手動取得）
  - DAU（日次アクティブユーザー）を相対0.38%向上（Duolingo実績）
  - 実装: localStorage内のstreak_freeze_countを管理

### 実装指針
```
- ストリークカウンター: 毎日のプレイを検知しインクリメント
- 表示: 炎アイコン＋日数表示（画面上部に常時表示）
- フリーズ: 3日連続プレイごとに1個自動付与（最大2個ストック）
- リカバリー: 途切れた翌日に「復活チャンス」を1回提供
- 節目祝福: 7日、30日、100日でアニメーション付き祝福
```

## 2. リーダーボードとリーグ制

### 心理学的メカニズム
- **社会的比較（Social Comparison Theory）**: 他者との相対的位置で自己を評価する欲求
- **相対的有能感**: 手の届く目標があるときにドーパミンが効率的に駆動

### 効果データ
- XPリーダーボード参加者: レッスン完了数40%増加
- ネガティブ経験報告: 無限リーダーボードで約19.8%

### マクロ vs ミクロ・リーダーボード
- **マクロ（全体ランキング）**: 上位以外に絶望感。避けるべき
- **ミクロ（階層別リーグ制）**: 推奨
  - 同程度の活動量のユーザー30人程度をグループ化
  - 週単位で昇格/降格を争う
  - 「少し頑張れば勝てる」現実的目標を維持

### 実装指針
```
- ローカルリーグ: localStorageに仮想対戦相手のスコアを生成
  - 自プレイヤーのスコア±10-30%の範囲でランダム生成
  - 10名程度の仮想リーグを構成
- 表示: 自分の順位を中心に上下3名を表示
- 昇降格: 週末にリーグをリセット、上位3名は昇格演出、下位3名は降格
```

## 3. アチーブメント（バッジ）

### 心理学的メカニズム
- **自己効力感（Self-Efficacy）**: 小さな成功体験の連続が「自分はできる」感覚を構築
- **スモールウィン**: 遠すぎる最終目標を細分化し、短期の達成感を連続提供
- **報酬予測誤差**: 予期しないバッジ付与がドーパミンを放出

### 効果データ
- バッジ導入: コース完了率30%向上

### バッジ設計原則
1. **行動に基づく**: 「10ゲームプレイした」等の具体的行動にリンク
2. **段階的解放**: 8回プレイ→ゲームごとの強み解放、25回→月間変動レポート（Lumosity方式）
3. **予期しない報酬**: 定期バッジに加え、条件非公開のシークレットバッジ
4. **社会的価値**: 取得バッジがプロフィールに表示され他者から見える

### 実装指針
```
バッジカテゴリ:
- 初回系: 「初めてのゲーム」「5人以上でプレイ」
- 継続系: 「7日連続」「30日連続」「100ゲーム達成」
- スキル系: 「処理速度上位10%」「全問正解」
- シークレット: 条件非公開（探索の楽しみ）

保存: localStorage内のbadges配列で管理
表示: 取得時にトースト通知＋アニメーション、プロフィール画面に一覧
```

## 外発→内発への移行設計
- **初期（Day 1-7）**: ストリーク＋バッジで外発的動機を形成（習慣の種まき）
- **中期（Day 8-30）**: リーグ制で社会的比較、成長ダッシュボードで有能感
- **長期（Day 31+）**: 社会的繋がり（マルチプレイ体験）と自律性（難易度選択）で内発的動機へ昇華
