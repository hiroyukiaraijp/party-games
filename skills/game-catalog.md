# ゲームメカニクスカタログ

## 概要
パーティゲームに適用可能な20のメカニクス一覧と、実装優先度TOP10を定める。

## 実装優先度 TOP10

| 順位 | ゲーム名 | 領域 | 選定理由 |
|------|---------|------|---------|
| **1** | ワードウルフ | 社会的知性 | 圧倒的リプレイ性。テキスト配列のみで高度な心理戦。画像アセット不要 |
| **2** | ストループバトル | 処理速度 | 実装が極めて容易。ストループ効果による誤答が万人の笑いを誘う |
| **3** | シンクロ・スケール | 社会的知性 | HTMLのrange入力で白熱議論が自然発生。世代問わず機能 |
| **4** | しりとり爆弾 | 言語力 | タイマー1つで強烈な焦燥感。ルール説明ほぼ不要 |
| **5** | 消えたのはどれ？ | 記憶力 | 絵文字配列から1つ抜くだけの実装。大人数で同時プレイ可能 |
| **6** | NGワード・スピーカー | 言語力 | アプリはお題出力のみ。実際のゲームは人間の会話に委ねる |
| **7** | フラッシュ暗算 | 処理速度 | DOM高速更新だけで完結。競技性高く短時間決着 |
| **8** | リアル・オア・ライ | 社会的知性 | 入力フォームのみ。プレイヤー自身がコンテンツアセット |
| **9** | ペア探し・フリップ | 記憶力 | 万人が知るルール。カード状態管理のみ |
| **10** | アナグラム錬金術 | 言語力 | 文字並び替えのアハ体験。D&D実装がやや手間 |
