Party Games / フラッシュメモリー

遊あそび方かた

  • グリッド上じょうにパターンが一瞬いっしゅんだけ表示ひょうじされます
  • パターンを記憶きおくして、タップで再現さいげんしてください
  • 正解せいかいセル数すうに応おうじてポイント獲得かくとく。完全一致かんぜんいっちでボーナス！
  • レベルが上あがるとセル数すうが増ふえて難むずかしくなります

鍛きたえられる力ちから

  • 空間認識くうかんにんしき — グリッド上じょうの位置関係いちかんけいを記憶きおくする力ちから
  • ワーキングメモリ — 短期間たんきかんパターンを保持ほじする力ちから
  • 注意力ちゅういりょく — 一瞬いっしゅんの表示ひょうじに集中しゅうちゅうする力ちから

プレイヤー

覚えてください！
回答:
記憶したパターンをタップ！
